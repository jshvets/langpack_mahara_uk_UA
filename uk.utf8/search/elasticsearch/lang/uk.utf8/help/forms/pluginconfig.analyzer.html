Analyseur d'Elasticsearch

L'analyseur d'Elasticsearch se charge d'analyser le contenu de la base de données de recherche d'Elasticsearch.

Maraha utilise le moteur d'analyse embarqué mahara_analyzer.

Lisez le manuel d'Elasticsearch si vous désirez utiliser un autre moteur d'analyse.

Ce paramètre ne peut pas être modifié via l'interface Web. Vous devez modifier le fichier config.php et changer le parmamètre :

$cfg->plugin->search->elasticsearch->analyzer.

Il n'est pas recommandé de modifier la valeur par défaut de ce paramètre, à moins que vous ne sachiez exactement ce que vous faites.
