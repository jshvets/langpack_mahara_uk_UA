Catégories de notifications

Vous pouvez choisir comment vous désirez être averti pour chaque catégorie d'activités.

Boîte de réception : la zone de notification affichera la liste des notifications reçues concernant les activités.

Courriel : votre adresse courriel principale recevra un message chaque fois qu'une activité a été réalisée.

Résumé par courriel : votre adresse courriel principale recevra un message par jour, résumant toutes les activités qui se sont déroulées durant les dernières 24 heures.

Flux d'activité : les activités apparaîtront sur la page où vous aurez placé un boîte de flux d'activités.

Note : si vous choisissez une des options du courriel, toutes les notifications seront aussi journalisées. Elles seront toutes déjà marquées comme ayant été lues. Ces notifications sont automatiquement supprimées du journal au bout de 60 jours
