Sources « iframe » autorisées Le code html entré par un utilisateur qui contient la balise <iframe> est supprimé systématiquement avant l'affichage de la page dans Mahara, à moins que la source du code à inclure figure dans la liste des sites autorisés. Pour réduire les risques d'une inclusion d'un code malveillant dans vos pages, par l'intermédiaire d'un « iframe », vous devez essayer d'être le plus précis possible dans la définition du code autorisé tout en restant suffisamment général pour inclure tous les documents nécessaires depuis la source que vous autorisez. Pour chaque source vous pouvez donc définir plusieurs instances de code « iframe » que vous voulez autoriser Quand vous ajoutez un nouveau site sur cette page, l'icône « favicon » est déterminée par le texte que vous avez entré et la base de l'URL est utilisée pour récurer l'icône sur le site distant. Vous pouvez aussi modifier cet icône après avoir ajouté un nouveau site, en cliquant sur le bouton d'édition.
