Étapes Behat

Vous pouvez avoir un aperçu des étapes Behat disponibles et où elles sont utilisées dans les tests. Tous les tests sont localisés dans le dossier de la base de code /test/behat/

Les variables à trouver dans les lignes de fonctionnalités sont représentées par des "?" pour grouper ces lignes plus facilement. Les lignes actuelles affichent les variables correctes.
