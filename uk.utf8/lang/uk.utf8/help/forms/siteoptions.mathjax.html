MathJax

MathJax un moteur d'affichage, écrit en javascript, pour la notation TeX, LaTeX, AsciiMath et MathMl qui permet d'écrire des équations scientifiques et mathématiques dans les pages de Mahara.

MathJax permet l'affichage des équations partout dans une installation Mahara.

URL de MathJax

La configuration par défaut utilise la dernière version stable du Content Distribution Network de MathJax. Les protocoles http et https protocols peuvent être utilisés.

Il est aussi possible de renseigner le paramètres, $cfg->mathjaxpath pour donner le chemin vers une version locale de MathJax. Consultez la documentation pour plus d'information à ce sujet.

La configuration par défaut permet d'utiliser aussi les notations MathML, AsciiMath, TeX or LaTeX et demander à MathML de les interpréter au format HTML-CSS.

Configuration avancée de MathJax

Des paramètres supplémentaires peuvent être ajoutés à la varibale $cfg->mathjaxconfig. La configuration par défaut de MathJax est suffisante pour la plupart des utilisateurs, mais MathJax peut être configurer de manière plus fine avec toutes les options possibles définies dans la documentation de MathJax.

Par défaut, l'extension mhchem qui ajoute la macro \\ce pour les formules chimiques est inclue. Consultez la documentation pour plus d'information.
