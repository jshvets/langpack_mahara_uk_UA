Icône de la licence

Vous pouvez utiliser les protocole http://... or https://... pour indiquer l'URL vers un image sur votre site ou extérieure à celui-ci.

Vous pouvez aussi utiliser une URL pour une licence, qui spécifie une icône particulière présente dans le thème de votre Mahara. Par exemple l'URL 

license:gfdl.png
permet d'accéder à l'icône 
static/images/license/gfdl.png
dans le thème.
