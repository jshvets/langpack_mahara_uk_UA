Navigation par menus déroulants

Choisir ici que la navigation s'effectue par menus déroulants n'implique que le thème pour cette institution. Si les utilisateurs peuvent librement choisir leur propre thème pour afficher les informations dans Mahara, cette option détermine pour chaque thème si les utilisateurs verront alors des menus déroulants ou des onglets pour naviguer dans le site.

Les thèmes imposés à tout un site prennent le dessus sur les préférences indiquées dans les réglages du site . Si les utilisateurs peuvent librement choisir un thème d'une autre institution pour afficher les informations dans Mahara, cette option détermine pour chaque thème si les utilisateurs verront alors des menus déroulants ou des onglets pour naviguer dans le site.
