Créer une page via les mots-clés

Vous pouvez sélectionner plusieurs mots-clés afin de placer automatiquement sur la page du contenu qui comprend ces mots-clés.

La page ainsi créée présentera le contenu trouvé (productions et blocs) avec les mots-clés indiqués. Ensuite, vous pouvez réarranger les informations sur la page comme vous le souhaitez..

Par exemple si vous marquez :

  • Une image et que vous sélectionnez le mot-clé là, elle sera ajoutée à la page dans un bloc image.
  • Trois images avec les mêmes deux mots-clés et que vous sélectionnez ces mots-clés là, les images seront ajoutées à la page dans une galerie d'images.
  • Un PDF et sélectionnez ce mot-clé là, le bloc PDF sera ajouté à la page.
  • Un bloc existant sur une autre page, par ex. : un bloc de type « Texte », il sera copié sur cette page.

Règle d'ajout à la page :

Type Nombre Bloc
Image 1 Image
2+ Galerie d'images
PDF 1 PDF
2+ Fichiers à télécharger
Fichier générique 1+ Fichiers à télécharger
Audio/vidéo 1 Média interne
2+ Fichiers à télécharger
Dossier 1+ Un bloc dossier pour chaque dossier
Journal 1+ Un bloc journal pour chaque journal
Article de journal 1+ Article de journal avec mot-clé
Notes 1+ Un bloc note pour chaque note
Profjet/tâche 1+ Un bloc projet par projet, soit l'ensemble du projet où se trouve la tâche marquée du mot-clé

Note : Les mots-clés sélectionnés ne sont utilisés que pour ajouter du contenu à une nouvelle page. Si vous désirez ajouter d'autres mots-clés par la suite vous devrez le faire vous-même sur la page. De même que, si vous supprimez un bloc sur la page, il ne reviendra pas automatiquement sur celle-ci.

Si vous sélectionnez un mot-clé différent (ou le même à nouveau) la fois prochaine, le contenu sera ajouté à nouveau à la page.
