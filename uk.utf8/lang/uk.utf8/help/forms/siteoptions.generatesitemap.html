Créer le plan du site

Cochez cette option si vous désirez que Mahara génère automatiquement un fichier pour les moteurs de recherche, leur indiquant la structure de votre site.

Ce document contient un ensemble de fichiers écrits en XML et suivant le format décrit par sitemaps.org. Le document du plan, « index », se trouve à l'URL [votresite]/download.php?type=sitemap. Il contient une liste de différents état du plan de votre site à travers le temps; il est mis à jour sur une base journalière. Chaque fichier ne contient que la liste des pages qui ont changé durant les dernières 24 heures
