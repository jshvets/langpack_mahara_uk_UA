Méthode de notification par défaut

La méthode utilisée pour envoyer des notifications pour les nouveaux utilisateurs. Si "Boîte de réception" est définie dans les préférences, les notifications sont conservées sur le site, et les utilisateurs doivent se connecter pour les lire sur leur Boîte de réception. Les utilisateurs peuvent modifier cette préférence à tout moment dans leur préférences de notification.
