Boîte de réception

Il s'agit d'une liste contenant toutes les activités système et les notifications qui vous ont été envoyées. Utilisez la page « Notifications » dans les « préférences » du compte, pour décider où vous désirez que ces informations soient envoyées. Une fois que vous avez lu une notification, vous pouvez la marquer pour suppression de votre liste.

Note : Les types de notifications suivants expirent automatiquement et sont supprimés de votre liste après 6 mois, si vous avez choisi l'option de l'envoi sur votre courrier électronique :

  • Notifications d'accès à une page
  • Notification de la liste de suivi
  • Messages des institutions
