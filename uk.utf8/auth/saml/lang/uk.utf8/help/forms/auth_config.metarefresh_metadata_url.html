URL vers les métadonnées pour le rafraichissement automatique

Si un URL valide est donné dans cette rubrique, le plug-in SAML téléchargera automatiquement les dernières métadonnées et les stockera localement. Ceci permet d'évider de devoir manuellement mettre à jour les métadonnées lorsqu'elles sont modifiées chez votre prestataire de service d'identité. Notez que les données récupérées à l'aide de cet URL remplaceront les métadonnées XML actuelles.
