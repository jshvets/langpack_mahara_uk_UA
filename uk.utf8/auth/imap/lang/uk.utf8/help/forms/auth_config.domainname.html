Nom de domaine

Seules les adresses électroniques appartenant à ce domaine sont acceptées. Cette option est intéressante si vous utilisez votre propre nom de domaine auprès d'un prestataire de service de messagerie électronique comme GMail ou Outlook, en utilsant le protocole IMAP.

Par exemple, si vous utilisez les services de GMail, vous pouvez entrer les informations suivantes :

Nom du serveur ou adresse IP : imap.gmail.com

Port : 993

Protocole : IMAP/SSL

Nom du domaine : VOTRE.DOMAINE
