Synchroniser les groupes de ces contextes seulement

Indiquez une liste de contextes qui comprennent les groupes. Separez chaque contexte par un « ; » (exemple : « ou=groups,o=org;ou=other,o=org »).

Si cette rubrique est laissée vide, le script cron de synchronisation utilisera la même liste de contextes que celle indiquée dans la rubrique « Contextes » où sont stockés les utilisateurs.
