Mode de forum avec retrait

Déplié : affiche tous les messages du forum dans l'ordre parent/enfant. Les réponses à un message sont ajouté comme enfant de ce message.

Déplié au maximum: Affiche les messages du forum dans l'ordre parent/enfant selon le maximum précisé dans le paramètre 'Nombre de niveaux de retrait'. Les réponses à un message sont ajouté commen enfant de ce message.

Aucun retrait : Affiche les message du forum dans l'ordre chronologique. Les réponses sont ajouté à la fin du fil de discussion.
