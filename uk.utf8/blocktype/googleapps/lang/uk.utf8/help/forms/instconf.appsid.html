Google Documents

Google Apps pour l'éducation : Le partage de documents individuels est semblable à la version standard d'un compte Google Apps.

L'administrateur de votre institution doit avoir configuré Google Apps de telle manière que seuls les membres de votre institution puissent voir vos fichiers partagés.
Dans ce cas, la personne qui accéderait une de vos Pages sur Mahara se verrait demander son nom d'utilisateur et son mot de passe pour afficher les documents stockés sur Google Documents; les fichiers inclus deviendraient alors visibles dans la Page.

Documents texte

  1. Ovvrir un Document texte sur Google Documents
  2. Cliquer sur Partager puis Publier sur le Web... (dans le coin en haut et à droite)
  3. Cliquer sur le bouton Démarrer la publication
  4. Copier le lien vers le document ou le code qui est affiché dans la rubrique Intégrer le code
  5. Coller le lien ou le code dans le bloc GoogleApps de Mahara

Collections

  1. Dans Google Documents cliquer sur Mes collections ou sur Collections partagées avec (dans la liste à gauche)
  2. Sélectionner la Collection que vous désirez inclure
  3. Cliquer sur le flèche à droite de la sélection puis choisir Partager > Partager
  4. Cliquer sur Modifier..., puis changer les Permissions de Privé à Public; cliquer sur Enregistrer
  5. Copier le lien depuis la rubrique Lien de partage
  6. Coller le lien dans le bloc GoogleApps de Mahara

Lecteur Google Documents

Cette fonctionnalité est la même que la fonction « Lecteur » de Gmail. Vous pouvez inclure et visualiser sur Mahara un fichier, qui est directement disponible sur Internet. Il vous suffit d'indiquer son URL complète (en incluant le nom du fichier) dans le bloc GoogleApps de Mahara.

Le Lecteur de Google Documents accepte les types de fichier suivants : PDF, TIFF, PPT, DOC, DOCX.

Note : Les fichiers que vous avez déposés dans la zone de dépôt de Mahara ne sont pas directement accessibles via une URL; vous ne pouvez donc pas les inclure de cette façon.

Google Agenda

  1. Ouvrir Google Agenda
  2. Cliquer sur Paramètres de Google Agenda (en haut à droite dans le menu)
  3. Cliquer sur le lien Agendas à gauche dans le menu du haut
  4. Cliquer sur le nom de l'agenda que vous désirez inclure
  5. Dans la rubrique Détails de l'agenda vous trouvez la rubrique Insérez cet agenda
  6. Personnaliser la présentation de l'agenda puis copier le code depuis la rubrique
  7. Coller le code dans le bloc GoogleApps de Mahara

Google Maps

  1. Ouvrir Google Maps
  2. Cliquer sur le lien Mes Adresses en haut à gauche
  3. Cliquer sur le nom de l'adresse que vous désirez inclure
  4. Cliquer sur l'icône du Link en haut à droite de la fenêtre
  5. Personnaliser la présentation de la carte, puis copier le code
  6. Coller le code dans le bloc GoogleApps de Mahara

Note : Suivez les étapes 2 et 3 seulement si vous désirez inclure un de vos Google Maps. Si par contre vous désirez inclure une carte quelconque, vous pouvez sauter ces deux étapes 2 et 3.

Google Livres

  1. Ouvrir Google Livres
  2. Rechercher le livre que vous désirez inclure
  3. Cliquer sur le lien ou le bouton Aperçu
  4. Cliquer sur l'icône du lien en haut à droite de la fenêtre
  5. Copier le code depuis la rubrique
  6. Coller le code dans le bloc GoogleApps de Mahara
